# animal-shelter

A small Nuxt app for browsing animals available for adoption. A personal learning project focused on Nuxt 4, Pinia, and component testing.

## Features
- Browse a list of animals and open a detail page for each
- State managed with Pinia
- Unit tests with Mocha and Chai

## Develop

```bash
bun install
bun dev
```

## Test

```bash
bun test
```

## Stack
Nuxt 4, Vue 3, Pinia, Tailwind CSS with daisyUI, Mocha and Chai.
